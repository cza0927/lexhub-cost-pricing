收费方案 两层利润点

与成本方案分开。① API 使用成本 → 设利润点；② 系统其他功能成本 → 各设利润点。对外按统一规则计价，后端实际跑哪个模型不影响对外价。

① 计费层一 · Token（API 使用）

统一计价规则：对外价 = 最高档位最贵模型 × 2。不论后端实际调用哪个模型，对外一口价；后端跑便宜模型（DeepSeek）即为利润。
